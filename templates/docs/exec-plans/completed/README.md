# Completed Exec Plans

Completed harness runs and standalone fixes live here.

Keep this directory useful for future agents:

- Preserve `meta.json`, `spec.md`, sprint contracts, build logs, and evaluations.
- Append post-completion fixes to the run's `hotfixes/hotfix-log.md`.
- Record standalone fixes in `standalone-fixes.md`.
- Append one-line summaries to `history.log`.
